Exercise 5:

Exercise 5: Write a function that takes a double named radius as input and returns a double which is the area of a circle with that radius. (hint: you can approximate π\piπ as 3.1412)

double CircleArea(double radius)
{
    double pi = 3.1412;
    return pi * radius * radius;
}
